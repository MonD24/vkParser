#  VK Contest Bot

Автоматический бот для участия в конкурсах ВКонтакте с уведомлениями в Telegram.

---

##  Возможности

-  **Поиск конкурсов** по ключевым словам через `newsfeed.search` (3 раза в день по расписанию)
-  **Автоматическое выполнение условий**  репост, лайк, подписка, комментарий
-  **Проверка победителей**  сканирует новые посты группы после окончания конкурса
-  **Авто-очистка**  удаляет репосты и отписывается от групп если не выиграл
-  **Форвард VK-сообщений** в Telegram в реальном времени
-  **Управление через Telegram**  команды `/start`, `/stop`, `/now`, `/status`, `/list`
-  **Умная фильтрация**  блокирует казино, крипто, маркетинг, юридические услуги, спам-агрегаторы

---

##  Требования

- Python 3.10+
- VK пользовательский токен (не группы!)
- Telegram бот токен + ваш chat_id

##  Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/MonD24/vkParser.git
cd vkParser
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv .venv
source .venv/bin/activate        # Linux/Mac
# или
.venv\Scripts\activate           # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить `.env`

```bash
cp .env.example .env
```

Открой `.env` и заполни:

```env
VK_TOKEN=vk1.a.ВАШ_ТОКЕН_ЗДЕСЬ
VK_USER_ID=123456789
TG_BOT_TOKEN=1234567890:ВАШТОКЕН
TG_CHAT_ID=123456789
```

**Как получить VK токен**  перейди по ссылке в браузере (залогинен как нужный пользователь):

Из адресной строки скопируй `access_token=...` до `&expires_in`.

**Как получить Telegram chat_id**  напиши [@userinfobot](https://t.me/userinfobot).

### 5. Запустить

```bash
python main.py --now   # запуск с немедленным первым циклом
python main.py         # только по расписанию
```

##  Деплой на Linux VPS

```bash
scp -r . root@ВАШ_IP:/opt/botwim/
ssh root@ВАШ_IP "bash /opt/botwim/deploy.sh"
```

Скрипт `deploy.sh` автоматически установит зависимости, создаст **systemd сервис с автозапуском** при перезагрузке и запустит бота.

### Управление сервисом:

```bash
systemctl status botwim        # статус
systemctl start botwim         # запустить
systemctl stop botwim          # остановить
systemctl restart botwim       # перезапустить
journalctl -u botwim -f        # логи в реальном времени
```

##  Фильтрация постов

Каждый найденный пост проходит несколько проверок:

1. **`is_contest()`**  минимум 2 конкурсных слова + условие участия + не итоги + не спам-агрегатор
2. **`is_blocked()`**  блок-лист: казино, крипто, 18+, маркетинг, юридические услуги...
3. **Гео-фильтр**  пост должен относиться к нужному региону
4. **Размер группы**  минимум `MIN_GROUP_MEMBERS` подписчиков
5. **`parse_conditions()`**  условия выполнимы автоматически (без фото/видео/кодовых слов/репоста в историю)

##  Дисклеймер

- Используй **пользовательский токен**, не групповой
- Бот делает паузы между действиями чтобы снизить риск ограничений от VK
- Используй на свой страх и риск в соответствии с правилами VK
